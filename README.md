# 🔥 FLAMES Game

A fun relationship compatibility game built with HTML, CSS, and JavaScript!

## Play Now

[Open the game](https://devselyan.github.io/flames-game/)

## Features

- 🎊 **Confetti animation** - Celebration burst when result is revealed
- 💫 **Particle effects** - Sparkles around the winning letter
- 📤 **Share button** - Copy your result to clipboard
- ⚡ **Fast animations** - Quick counting with dramatic reveals
- 📱 **Responsive design** - Works on mobile and desktop

## How to Play

1. Enter your name
2. Enter your partner's/crush's/friend's name
3. Click "Check FLAMES"
4. Watch the dramatic elimination process
5. Discover your relationship!

## FLAMES Meaning

- **F** - Friends
- **L** - Love
- **A** - Affection
- **M** - Marriage
- **E** - Enemy
- **S** - Siblings

## Local Setup

1. Clone this repository
2. Open `index.html` in your browser

```bash
git clone https://github.com/Devselyan/flames-game.git
cd flames-game
open index.html
```

## Deploy to GitHub Pages

1. Go to your repository Settings → Pages
2. Select source as `main` branch
3. Your site will be live at `https://devselyan.github.io/flames-game/`
